Localización fiscal de El Salvador 🇸🇻
Extiende las funcionalidades contables para adaptarlas a los requerimientos fiscales y contables de El Salvador. Proporciona características específicas para cumplir con las regulaciones locales y facilitar la gestión contable en empresas salvadorenas.

Descripción general

    Catálogo de cuentas

    El árbol del catálogo de cuentas contables es una estructura jerárquica que más comunes están disponibles dentro de su grupo. Use los grupos como referencia si quiere crear una cuenta nueva

    Impuestos

    Agrega impuestos básicos para poder emitir DTE, como el IVA, EXENTO, EXPORTADOR, IMPORTADOR y otros impuestos

    Catálogos:

    Integro los siguientes catálogos publicados por el ministerio de Hacienda V1.1 - 08-2024

    • CAT-019: Actividad Economica
    • CAT-029: Tipo de persona
